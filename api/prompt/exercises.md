# Referenz: Übungsfamilien

## Struktur

Jede Übung gehört zu einer **Familie** (Bewegungsmuster) und hat eine **Stufe** (Komplexität/Koordinationsanspruch).

**Stufe 1** = geführte Bewegung (Maschine, Kabelzug, einfache KH-Übungen). Beginner starten hier.
**Stufe 2** = freie Bewegung, moderat (Kurzhanteln, komplexere Kabelübungen). Ab ~3–6 Monate.
**Stufe 3** = freie Bewegung, komplex (Langhantel-Grundübungen). Ab ~6+ Monate oder mit Vorerfahrung.

**Wichtig:** Stufe 1 ≠ nur Maschinen. Kabelzug und einfache KH-Übungen (z.B. KH-Curl, Seitheben)
gehören auch zu Stufe 1, weil der Koordinationsanspruch gering ist.

## Regeln für Übungsauswahl

### Stufenzuordnung nach Level und Equipment

| Level | Equipment | Primäre Stufe | Erlaubt |
|-------|-----------|---------------|---------|
| `beginner` | Maschinen + Kabelzug | Stufe 1 | Alle Stufe-1-Übungen (inkl. Kabel + einfache KH) |
| `beginner` | Nur Freihantel | Stufe 1–2 | Einfache KH-Übungen, Goblet Squat, KH-Rudern |
| `intermediate` | Beliebig | Stufe 2 (Stufe 3 mit Erfahrung) | Alle KH + einfache LH (RDL, Rudern) |
| `advanced` | Beliebig | Stufe 3 bevorzugt | Alle Übungen verfügbar |

### Pro Split-Tag (Reihenfolge = Pflicht, siehe unten)
- **GK (Ganzkörper):** 1× Kniebeuge-Muster + 1× horizontales Drücken + 1× horizontales Ziehen + 1× Hüftstreckung + 1× vertikales Drücken oder Ziehen + ggf. Isolation + Core (= 5–7 Übungen je nach Zeit)
- **OK (Oberkörper):** Bankdrücken/Brustpresse + Rudern + Schulterdrücken + Latzug/Klimmzug + Trizeps + Bizeps (= 6 Übungen). Optional: Seitheben, Rear Delt.
- **UK (Unterkörper):** Kniebeuge + Kreuzheben/RDL + Ausfallschritte/Beinstrecker + Beinbeuger + Waden + Core (= 6 Übungen)

### Reihenfolge innerhalb einer Einheit

**Prinzip: ZNS-Last absteigend, Komplexität absteigend.**
Schwere Mehrgelenk zuerst (frisches Nervensystem, beste Technik), Isolation am Ende.

**GK (Ganzkörper) — Reihenfolge verbindlich:**
1. Schwere Mehrgelenk Unterkörper (Beinpresse/Kniebeuge)
2. Schwere Mehrgelenk Oberkörper horizontal (Brustpresse/Bankdrücken)
3. Schwere Mehrgelenk Oberkörper Ziehen (Rudern)
4. Hüftstreckung (Beinbeuger/RDL)
5. Vertikales Drücken oder Ziehen (Schulterdrücken/Latzug)
6. Isolation (Bizeps, Trizeps, Waden, Seitheben)
7. Rumpf/Core (Plank, Pallof Press)

**OK (Oberkörper) — Reihenfolge verbindlich:**
1. Horizontales Drücken (Bankdrücken/Brustpresse) — höchste Last
2. Horizontales Ziehen (Rudern) — schwer, Mehrgelenk
3. Vertikales Drücken (Schulterdrücken)
4. Vertikales Ziehen (Latzug/Klimmzug)
5. Isolation: Seitheben, Rear Delt (optional)
6. Isolation: Trizeps
7. Isolation: Bizeps

**UK (Unterkörper) — Reihenfolge verbindlich:**
1. Kniebeuge-Muster (Kniebeuge/Beinpresse) — höchste ZNS-Last
2. Hüftstreckung (Kreuzheben/RDL) — zweithöchste ZNS-Last
3. Einbeinig/Assistenz (Ausfallschritte, Beinstrecker, Beinbeuger)
4. Waden
5. Rumpf/Core

**Warum diese Reihenfolge nicht verhandelbar ist:**
Kniebeuge nach Bizeps-Curls = ermüdetes ZNS bei der komplexesten Übung.
Kreuzheben am Ende = Rücken vorermüdet, Technik bricht ein.
Isolation zuerst = Zielmuskeln vorermüdet, Mehrgelenk-Performance sinkt.

### Aufwärmsätze (für Beginner anzeigen)
Vor jeder Mehrgelenk-Übung:
- Satz 1: ~50 % Arbeitsgewicht × 10 Reps
- Satz 2: ~70 % Arbeitsgewicht × 5 Reps
- Dann Arbeitssätze

Isolation-Übungen: 1 leichter Aufwärmsatz reicht.

### Variation über Einheiten (Übungstausch innerhalb der Familie)

Bei 3× GK/Woche: Die drei Einheiten sollten NICHT identisch sein.
Variation innerhalb der Familie über die Wochentage:

Beispiel GK Beginner (Maschine + Kabel):
- **GK-A:** Beinpresse / Brustpresse / Kabelrudern eng / Beinbeuger liegend / Latzug breit / Seitheben KH / Plank
- **GK-B:** Hack-Squat / Kabel-Flys + Schulterpress-Maschine / Kabelrudern breit / Kabel-Pull-Through / Latzug eng / Bizeps-Curl Kabel / Kabelzug-Crunch
- **GK-C:** Beinpresse / Brustpresse / Rudermaschine / Beinbeuger sitzend / Face Pulls / Trizeps Kabel / Pallof Press

So trainiert der Beginner alle Bewegungsmuster, aber mit Variation über die Woche.
